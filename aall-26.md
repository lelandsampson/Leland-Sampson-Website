### One-stop platforms

Build and deploy in one place

* [Replit](https://replit.com/)
* [Vibe-code and deploy in the browser — no local setup needed. Good for quick projects and prototypes; pricing can scale with use.](https://replit.com/)
* [Lovable](https://lovable.dev/)
* [Similar approach to Replit: build a full web app in the browser and deploy with one click. Strong for rapid prototyping without touching a terminal.](https://lovable.dev/)

### Developer stack

Maximum flexibility — requires installing a few tools

* [Claude Code— Anthropic](https://claude.com/claude-code)
* [AI coding assistant that runs on your computer. Handles multi-file projects and iterative debugging — more capable than the all-in-one platforms for serious work.](https://claude.com/claude-code)
* [GitHub](https://github.com/)
* [Free code hosting. Connects your project to Render so pushing an update refreshes your live site, and gives you a history so you can undo mistakes.](https://github.com/)
* [Render](https://render.com/)
* [Free-tier web hosting. Connects to GitHub and redeploys on push. Works with Canvas exports or full Claude Code projects.](https://render.com/)
